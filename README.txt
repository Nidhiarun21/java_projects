Contact Management Application
This is a prototype of a contact management application that allows you to create and manage contact books, add and edit contacts, list contacts, search for contacts, and more.

This prototype is designed to demonstrate the functionality of a contact management application. It allows users to create and manage contact books, add and edit contacts, list contacts in various ways, and search for specific contacts based on various criteria.

Features
Create and manage contact books.
Add, edit, and remove contacts within contact books.
List contacts sorted by name, tags, creation date, or string length.
Search for contacts using a search string.
File-based storage for contact data.

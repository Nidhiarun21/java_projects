Contact Management Application
This is a prototype of a contact management application that allows you to create and manage contact books, add and edit contacts, list contacts, search for contacts, and more.

This prototype is designed to demonstrate the functionality of a contact management application. It allows users to create and manage contact books, add and edit contacts, list contacts in various ways, and search for specific contacts based on various criteria.

Features
Create and manage contact books.
Add, edit, and remove contacts within contact books.
List contacts sorted by name, tags, creation date, or string length.
Search for contacts using a search string.
File-based storage for contact data.
Getting Started
Prerequisites
Java Development Kit (JDK)
An Integrated Development Environment (IDE) is recommended for development.
Installation
Clone the repository to your local machine:
bash
Copy code
git clone https://github.com/yourusername/contact-management-prototype.git
Open the project in your preferred IDE.

Build and run the project to launch the application.

Usage
Main Menu
When you run the application, you will be presented with a main menu that provides various options for managing contacts:

Press 1 to Create Contacts Book
Press 2 to Load Contacts Book
Press 3 to Search contacts
Press 4 to List contacts
Press 5 for Birthday reminders
Press 6 to Exit
Creating a Contacts Book
If you select option 1 to create a contacts book, you will be prompted to enter a unique name for the book. The application will validate the name to ensure it doesn't already exist.

After successfully creating a contacts book, you will be presented with a submenu that offers options to:

Press 1 to Add a Contact
Press 2 to Edit a Contact
Press 3 to Remove a Contact
Press 4 to List the Contact
Press 5 to Search Contact
Press 6 to Go back
